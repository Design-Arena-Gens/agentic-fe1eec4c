# Agentic Minimalistic Dashboard

A clean, minimal dashboard for day-to-day life built with Next.js and Tailwind CSS. Widgets include time, weather (Open-Meteo), todos, notes, bookmarks, and a Pomodoro timer. All personal data stays in your browser via localStorage.

## Development

- Install dependencies: `npm install`
- Run dev server: `npm run dev`
- Build: `npm run build`
- Start: `npm start`

## Deployment

Vercel CLI can deploy with `vercel deploy --prod`.
